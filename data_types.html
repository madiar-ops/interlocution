Основные типы данных

Типы данных в Python — это классификация, определяющая значения переменной или объекта, а также какие операции можно с ними выполнить. В Python есть разные типы данных, каждый из которых обладает своими уникальными характеристиками и возможностями использования:В Python есть несколько основных типов данных:

  • int — целые числа
  • float — числа с плавающей запятой
  • str — строки
  • bool — логические значения
  • list — списки

Пример:

# Пример работы с типами данных
age = 25
name = "Иван"
is_student = True
print(f"Имя: {name}, Возраст: {age}, Студент: {is_student}")
