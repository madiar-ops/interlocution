Что такое циклы?

Циклы — базовый инструмент программирования на Python. С их помощью разработчики могут быстро выполнять повторяющиеся действия, автоматизируя процесс. В языке Python есть два вида циклов: Цикл for используется, когда известно, сколько раз нужно повторить блок кода. Он работает со строками, списками, файлами, словарями и множествами. Цикл while используется для воспроизведения блока кода с неизвестным числом повторений. После ключевого слова while указывается условное выражение, и пока это выражение возвращает значение True, будет выполняться блок инструкций, который идёт далее.

Пример:


        for i in range(10, 5, -2):
        print(i)
  
  Здесь используется вызов метода range с тремя параметрами.
  Третий параметр задаёт шаг итерации, то есть определяет, сколько элементов будет пропущено при перемещении от начала до конца диапазона.
  По умолчанию шаг равен 1 и его можно не указывать. В примере указано значение −2, то есть каждое повторение цикла будет смещать начальную позицию на 2 элемента назад.
  В итоге после выполнения программы в терминале будет последовательность «10, 8, 6».
